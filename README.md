# Audio Teapot
VRChatの音声入力にいろいろな音声を注入するWindowsアプリです。

# 使い方
AudioTeapot.exeを起動するだけ。

# 機能
## 囁きアビリティ
[memexさんの囁きアビリティ](https://memex.booth.pm/items/1177208)を発動します。

約20Hzのサイン波を常に出力してVRChatのノイズゲートを無効化することで、楽器演奏を途切れずに視聴者へ届けることができます。
